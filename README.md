# Тестовое задание DevOps
Реализация простого веб-приложения на Python за реверс-прокси Nginx в Docker-контейнерах.

## Как запустить
Для сборки и запуска всех компонентов выполните команду:
```bash
docker-compose up -d --build
```
## Как проверить
Чтобы убедиться в работоспособности, выполните запрос через терминал:
```bash
curl http://localhost
```
Ожидаемый ответ: Hello from Effective Mobile!.

## Описание архитектуры

Nginx cлушает порт 80 и проксирует все запросы на внутренний сервис backend. Python-сервер на базе http.server, работающий на порту 8080 внутри изолированной docker-сети. Приложение запускается от имени пользователя без прав root. Бэкенд недоступен напрямую с хоста.

## Используемые технологии
- Docker
- Docker Compose
- Python 3.12-slim.
- Nginx Alpine.
